# Quant Practice - 量化练习题仓库

本仓库将用于存放量化金融相关的练习题及解答，覆盖量化数据清洗、算法实现、数理金融、编程实践等多个方向，适合量化入门学习、进阶练习以及备考使用。

## 仓库地址
🔗 仓库链接：[https://github.com/nichengyu/quant_pratice](https://github.com/nichengyu/quant_pratice)

## 环境依赖
大部分代码基于 Python 实现，核心依赖包：
```bash
# 推荐使用conda/pip安装
pip install numpy pandas scipy matplotlib scikit-learn backtrader pyfolio
```

## 如何使用
1. 克隆仓库到本地：
```bash
git clone https://github.com/nichengyu/quant_pratice.git
cd quant_pratice
```
2. 按目录分类查看习题，每个习题文件包含：
   - 题目描述
   - 解题思路/公式推导
   - 代码实现（如有）
   - 结果分析（如有）
3. 可基于仓库习题进行拓展练习，欢迎提交自己的解答或新题目。
